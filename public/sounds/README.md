# Sound Files for Bracket Reveal

This directory should contain sound files for the bracket reveal feature.

## Required Files:

1. **reveal.mp3** - Sound played when revealing a match
   - Duration: 1-2 seconds
   - Type: Short, exciting sound effect
   - Format: MP3

## How to Add Sound Files:

1. Place your sound files in this directory
2. Ensure the file is named `reveal.mp3`
3. The sound will automatically play when revealing matches

## Alternative Sound Options:

You can also use:
- `reveal.wav` - WAV format
- `reveal.ogg` - OGG format

## Sound Recommendations:

- Use short, exciting sound effects
- Keep file size small (< 100KB)
- Test volume levels for streaming
- Consider copyright-free sounds

## Free Sound Resources:

- Freesound.org
- Zapsplat.com
- SoundBible.com
- Pixabay Sound Effects

## Note:

If no sound file is found, the reveal will still work but without audio.
